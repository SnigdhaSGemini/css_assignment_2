Difference Between : rem , em , vh , vw , px

Absolute length units

px :: Pixels 1px = 1/96th of 1in

Relative length units

(For Font Size)

em :: Font size of the parent, in the case of typographical properties like font-size (parent)

and font size of the element itself, in the case of other properties like width. (child)

rem :: Font size of the root element. (this size is relative to root element)

(For Dimensions)

vw :: 1% of the viewport's width.

vh :: 1% of the viewport's height.



Chaining Concept

Type 1 ::

(Space seperated)

It matches all elements that are descendants of a specified element.

Type 2 ::

(Comma Seperated)

It matches all elements mentioned in it.
